Какая фукциональность нужна

1. запуск отдельно nats
 при этом параметры - порт, количество очередей. Автоматом запускается мониторинг длины очередей (допустим раз в минуту)
1. запуск продусера - порт nats, количество очередей - раз в минуту - количество произведенного
1. запуск ноды - порт nats, количество очередей, бутстрап или нет - раз в минуту - количество обработанного

mode=[nats, producer, subscriber]
nats - port, queuesnum
producer - natsport, queuesnum
subscriber - natsport, queuesnum, bootstrap
